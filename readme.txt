== Description ==

Just a simple popup fixed at bottom of the screen. You can add content in a wusiwug editor and use it for your cookies notice.
 

== Installation ==

1. Upload the entire `simple-cookies-notice` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Simple cookies notice' menu under WordPress admin panel Settings.
